Math

    1、Math.sqrt()   函数返回一个数的平方根
    2、Math.cbrt() 函数返回一个数的立方根
    3、Math.trunc() 函数返回数值的整数部分
    4、Math.sign() 判断数值是正数还是负数
    5、Math.abs() 函数返回一个数的绝对值
    6、Math.acos() 函数返回一个数的反余弦值
    7、Math.asin 函数返回一个数的正余弦值
    8、Math.ceil（） 向上取整。
    9、Math.floor() 向下取整。
    10、Math.cos 返回一个数的余弦。
    11、Math.exp()  方法可返回 e 的 x 次幂的值。 e 代表自然对数的底数，其值近似为 2.71828。
    12、Math.log() 返回自然数的对数。（底为e）。
    13、Math.max() 返回参数中的最大数。
    14、Math.min() 返回参数中的最小数。
    15、Math.pow(x, y) 返回x的y次幂。
    16、Math.random（） 返回0-1之间的随机数。
    17、Math.found() 返回四舍五入后最接近的正数。
    18、Math.valueOf() 返回math对象的原始值。
    
Array

    19、Array.from(小， fun)  可以把伪数组转换成真正的数组， 第二个参数是个函数，非必填，可以对数组里的每一个进行元算。
    20、Array.of(x, y, z, ...) 可以将一组值转换成一个真正的数组。
    21、Array.copyWithin() 数组实例的copyWithin()方法，在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），
            然后返回当前数组。也就是说，使用这个方法，会修改当前数组。
            它接受三个参数。
                target（必需）：从该位置开始替换数据。如果为负值，表示倒数。
                start（可选）：从该位置开始读取数据，默认为 0。如果为负值，表示从末尾开始计算。
                end（可选）：到该位置前停止读取数据，默认等于数组长度。如果为负值，表示从末尾开始计算。
            [1, 2, 3, 4, 5].copyWithin(0, 3)  // [4, 5, 3, 4, 5]
    22、find  用于找出第一个符合条件的数组成员。它的参数是一个回调函数，所有数组成员依次执行该回调函数，
              直到找出第一个返回值为true的成员，然后返回该成员。如果没有符合条件的成员，则返回undefined。
              [1, 5, 10, 15].find(function(value, index, arr) {
                return value > 9;
              }) // 10
              function f(v){
                 return v > this.age;
              }
              let person = {name: 'John', age: 20};
              [10, 12, 26, 15].find(f, person);    // 26
              上面的代码中，find函数接收了第二个参数person对象，回调函数中的this对象指向person对象。
    23、findIndex  返回第一个符合条件的数组成员的位置，如果所有成员都不符合条件，则返回-1。
    24、fill方法使用给定值，填充一个数组。 ['a', 'b', 'c'].fill(7) // [7, 7, 7]
    25、keys(),values() keys()返回所有的key值，values()返回所有的value.
    26、includes方法返回一个布尔值，表示某个数组是否包含给定的值，与字符串的includes方法类似
    27、flat() 用于将嵌套的数组“拉平”，变成一维的数组。该方法返回一个新数组，对原数据没有影响。
        [1, 2, [3, 4]].flat()  // [1, 2, 3, 4]
        flat(参数)可以传入参数，Number，代表可以展开基层， 如果不管有多少层嵌套，都要转成一维数组，可以用Infinity关键字作为参数。
        [1, [2, [3]]].flat(Infinity) // [1, 2, 3]
